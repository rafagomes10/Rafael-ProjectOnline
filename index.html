GastroMenu - Advanced Gastronomy

Tecnologias: Next.js 15, React 19, TypeScript, TailwindCSS, Font Awesome.

O GastroMenu foi pensado como um micro SaaS, onde boa parte do layout e das cores são carregados a partir de um JSON mock. A ideia é que, futuramente, esse JSON seja gerado e editado por um painel administrativo. Nesse painel, os clientes poderão personalizar o layout e gerenciar os cardápios adicionando, editando ou removendo itens, tudo de forma simples e prática. Ou seja, basicamente eles vão editar através do painel administrativo, esse json que o frontend foi desenvolvido em cima. O frontend foi construído de forma dinâmica para se adaptar às alterações feitas nesse JSON.

    Funcionalidades
  • Menu Navegável por Categorias e Subcategorias
  • Layout personalizável através de um painel administrativo
  • Gerenciamento dos itens do cardápio: adicionar, editar e remover itens através de um painel administrativo
  • Listagem de Produtos
  • Modal de Detalhes do Produto
  • Compartilhamento de Produtos
  • Sincronização com a URL
  • Acessibilidade
  • Design responsivo para desktop e mobile
